Desarrollo web.

Si quieres aprender desarrollo web, estás en el sitio indicado. Hemos encontrado para ti los mejores recursos online y los hemos organizado según su dificultad y su temática. Comenzaremos con una introducción, e iremos avanzando poco a poco. El curso está diseñado para ser completado en orden (a no ser que ya domines algo), empezando desde la parte de arriba, hasta abajo.

Aprenderemos una serie de tecnologias que te permitirán crear cualquier tipo de aplicación. Esta base te servirá para más tarde continuar con el aprendizaje por tu cuenta de otros lenguajes y tecnologias. El objetivo principal es que te hagas una idea sobre cómo funciona todo.

1: Lo básico

La web, Internet, ¿Qué hace un desarrollador web?, ¿Qué tipos de trabajo existen?. Empezaremos contestando a las preguntas básicas que se te puedan ocurrir. ¡Entenderás dónde te estás metiendo!

2: Front-End

En esta parte nos centraremos en el Front-End. Aprenderemos lenguajes como HTML, CSS y Javascript. Además, aprenderemos a subir nuestra web a internet. Esto es más o menos lo basico para crear una web estática. Después de esto estaremos preparados para crear webs dinámicas.

2.1: Frameworks y Git

Después de aprender lo anterior, nos centraremos en unas herramientas llamadas frameworks que nos ayudarán a diseñar nuestras web de una forma mucho más rápida. También aprenderemos Git, una herramienta básica para todo desarrollador.

3: Back-End

Now that you've got a good grounding in Ruby, it's time to put that to work by learning how to rapidly develop websites using Ruby on Rails. By the end of this course, you'll be able to build and deploy a real website with confidence. In fact, you'll do it many times.
