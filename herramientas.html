Terminal de Comandos. Herramienta.

La comunmente conocida como consola o terminal, es una de las herramientas más intimidantes que hay en el mundo de la informática. Sin embargo, una vez comprendido el funcionamiento básico y sus infinitas posibilidades, serás capaz de realizar tareas complicadas de una forma rápida y potente gracias a unos pocos comandos.

¿Cómo empezar?

Todos los Pc's incluyen un terminal, por lo que no será necesario instalar nada. Sólo hay que saber dónde encontrarlo:

Mac: Finder > Aplicaciones > Utilidades > Terminal
Windows: Inicio > Ejecutar... > cmd
Linux: Si usas Linux, ya sabrás dónde buscar.

Lo habrás hecho bien si ves una pantalla que te invita a escribir a partir del símbolo $

Git. Control de Versiones.

Creado por Linus Torvals. Git es un programa imprescindible para no volverte loco. Quieras o no, como desarrollador, en algún momento deberás aprender algún sistema de control de versiones para gestionar tus proyectos. Muy unido a él está el popular Github, que no es más que un hosting de repositorios creados con Git.

¿Cómo empezar?

Lo primero es lo primero. Aunque se puede trabajar con Git de una forma gráfica, es casi imprescindible tener unos conocimientos mínimos del terminal. Si es así, ¡adelante!:

1- Ve a la web de Git y descárgate el software para tu sistema operativo.
1.1- No es obligatorio hacerse una cuenta en Github, pero te aclarará bastante las cosas.
2- Ármate de paciencia y sigue las guias, al principio será complicado, pero después verás que es bastante sencillo.

Sublime Text. Editor de texto.

Haz diez cambios a la vez, no un cambio diez veces consecutivas. Sublime es uno de los editores de texto más potente y popular entre los desarrolladores. Es multiplataforma y soporta un montón de plugins. Ojo, no es gratis, pero existe una versión de prueba que dura eternamente.
