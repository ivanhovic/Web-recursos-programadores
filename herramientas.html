Terminal. Herramienta.

Seguramente lo habrás visto en alguna película. Un tio aporreando el teclado mientras una ventana negra del ordenador se llena de letras verdes de las que no entiendes una palabra... Esa ventana es el terminal de comandos, comunmente conocido como consola o terminal, una de las herramientas más intimidantes que hay en el mundo de la informática.

Git. Control de Versiones.

Creado por Linus Torvals, Git es una herramienta imprescindible para no volverte loco. Quieras o no, como desarrollador, en algún momento deberás aprender algún sistema de control de versiones para gestionar tus proyectos. Muy unido también al popular Github, que no es más que un hosting de repositorios creados con Git.

Sublime Text. Editor de texto.

Sublime es uno de los editores de texto más populares actualmente. Es multiplataforma y soporta un montón de plugins. Ojo, no es gratis, pero existe una versión de prueba que dura eternamente.

Wordpress. Hosting.

Además de para hacer el típico blog en Wordpress.com, existe también otra versión algo más, como decirlo, profesional, llamada Wordpress.org. Aquí deberás instalar la aplicación en tu servidor, buscarte un dominio... Gran parte de la web está hecha así.

Bootstrap. Framework.

Si, como hemos dicho, gran parte de la web esta hecha con Wordpress, la otra mitad está hecha con Bootstrap (como esta misma).
