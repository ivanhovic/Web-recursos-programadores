El Front-End

Ahora que ya sabes qué hacen los desarrolladores web, es el momento de ensuciarse las manos y hacer lo que hacen ellos. En esta primera parte aprenderás los fundamentos de la programación web. Aprenderás a construir una web sencilla con HTML, darle estilo con CSS y añadir algunos elementos interactivos con JavaScript. Puede que aún no entiendas nada, pero no te preocupes, irás comprendiéndolo poco a poco.

Básicamente esto es el llamado Front-End. Haciendo una analogía, si una web fuera un coche, el Front-End se encargaría de poner el color, los asientos, los botones que controlan la radio... Para hacer el motor, necesitaríamos el Back-End.

0. Configuración

Antes de empezar necesitaremos configurar nuestro entorno de trabajo. Para esta primera parte necesitaremos un editor de texto y un navegador web. El primero para programar y el segundo para ver los resultados.

Editor de texto: Sublime Text

Haz diez cambios a la vez, no un cambio diez veces consecutivas. Sublime (así lo llamamos los amigos), se ha convertido en uno de los editores de texto más potentes y popularers entre los desarrolladores. Es sencillo, muy rápido, multiplataforma y soporta un montón de plugins. Ojo, no es gratis, pero existe una versión de prueba que dura eternamente.

Vale, ¿y para que necesito un editor de texto?, ¿no me vale con el Word? No, a partir de ahora escribirás código, no texto. Word está preparado para texto y Sublime está preparado para programar. Si no te gusta Sublime, o simplemente prefieres otro editor, al menos comprueba que tenga la capacidad de colorear el código.

Navegador: Chrome o Firefox

Para empezar nos valdría con cualquier navegador (Chrome, Firefox, Safari, Opera, Explorer), sin embargo Google y Firefox son los que llevan la delantera en cuanto a adaptar los nuevos estándares de la web a sus navegadores. Por ello y por su buen rendimiento, nos decidimos por uno de los dos.

Ejercicio

  1. Abre Sublime y escribe: ¡Mi primera Web!.
  2. Guarda el archivo con este nombre: index.html.
  3. Abre index.html en tu navegador favorito.

Si tu resultado es este, ¡estás preparado para el siguente nivel!

1. HTML y CSS

El HTML es el primero de los tres lenguajes que deberás aprender para crear webs. Con él se definen todos los elementos que ves: el texto, las fotos, los videos, o las listas. También proveen a los navegadores con otros archivos como el CSS y el JavaScript.

El segundo de los lenguajes es el CSS (Cascade style sheets), que define el estilo de una web: colores, tipografia, tamaños, espacios e incluso algunas animaciones.

Haciendo una analogía, un arquitecto haría los planos de una casa y la construiría con el HTML, más tarde, un decorador la haría más bonita con el CSS.
