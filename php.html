PHP

Posiblemente te habrás preguntado qué es eso de las webs dinámicas o cómo cambian constantemente algunos datos de las páginas web. Uno de los que tienen la culpa de eso es PHP.

PHP es un lenguaje de programación de uso general diseñado especifícamente para el desarrollo web. Por ello, irá directamente incorporado dentro del documento HTML.

Es importante comprender que PHP se ejecuta en el servidor, es decir, que no se ejecuta en tu ordenador, sino en uno remoto.

El navegador puede enternder HTML y CSS, pero necesitará algo de ayuda para el PHP. Para ello necesitaremos instalar un servidor en nuestro ordenador(normalmente Apache).
